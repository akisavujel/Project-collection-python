# 🤖 FaceDetection-OpenCV

A simple Python project that performs **real-time face detection** using OpenCV and your webcam.   Detects faces, draws rectangles around them, and works smoothly on macOS and Windows. 💻📹

## ✨ Features

- 👁️ Real-time face detection using Haar cascades  
- 👥 Detects multiple faces in a frame  
- 🟩 Draws rectangles around detected faces  
- ⏹️ Press **'a'** to exit the program  
- ⚡ Optimized for performance with frame resizing
